== Installation ==

Installing and activating the plugin is really easy. You should be able to configure the plugin in a few easy steps.

1. To get started, please enter your <a href="https://platform.openai.com/account/api-keys" target="_blank"> OpenAI API key </a> in the dashboard configuration settings. 
2. Once completed, head to the Settings page to customise and configure the AI Modules you'd like to implement on your website.
3. Configure the settings for each of the activated AI modules. 

For additional guidance, browse our detailed tutorials, or feel free to reach out if you have any questions. Let's unleash the power of AI together!

== Open AI API Configuration ==

Please configure the Press AI plugin by entering OpenAI API on your dashboard configuration settings. You can get the <a href="https://platform.openai.com/account/api-keys" target="_blank"> OpenAI API key on this page</a> once you have signed up and are logged into your account. 

You can see the complete instructions to sign up and get a free API key from the Open AI Website on the plugin dashboard.

== Frequently Asked Questions ==

= What does the Press AI Plugin do? =
Press AI WordPress plugin helps you integrate Generative AI tools, including content generators, AI writers, and chatbots, into your website, enhancing creativity and user engagement.

= How do I install and activate Press AI Plugin? =
Plugin Installation is straightforward. You can either download and upload the plugin to your WordPress website or directly install it by searching with your Plugins tab of WordPress dashboard. Active the plugin and you can then configure it by entering your OpenAI API key and customizing the AI modules in your plugin dashboard settings.

= Where can I obtain an OpenAI API key? =
You can sign up and obtain a free API key from the OpenAI website <a href="https://platform.openai.com/account/api-keys" target="_blank"> Get OpenAI API key here</a>. Detailed instructions are also available on your plugin dashboard.

= Is Press AI compatible with all WordPress themes? =
Press AI is designed to be compatible with most WordPress themes. We have tested the plugin with all major WordPress themes and should work without any problems. Feel free to tweak the chatbot settings if you don't like the default look. 

= Can I customise the AI-generated content? =
Yes, you can customise the settings for each AI module to tailor the generated content according to your website's needs.

= What kind of support is available for Press AI users? =
We offer detailed tutorials for setup and use. Additionally, you can reach out to us directly for any specific queries or support needs.
